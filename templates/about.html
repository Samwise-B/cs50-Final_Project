About

For my final project I decided to build on what we have learnt on the course in order to create a simple flask web app game. The idea behind the game is to learn about potential security flaws that occur when developing a dynamic flask application. The "hacking" puzzle game, is aimed at people like myself, who are novice programmers that are interested in learning web development and a few basic web application security vulnerabilities. This puzzle game is a simple representation of actual security flaws that you could see in the wild designed in a way that is hopefully fun and educational.

Requirements

The game is designed to be easy enough for anyone new to computer science to complete. If you completed CS50 you will most likely have the skills needed to beat the game. Hopefully you should also be able to complete the entire game without having to use all sorts of complicated pentesting tools.
