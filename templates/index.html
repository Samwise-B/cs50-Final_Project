Final Project

This web application is a hacking game. There are three levels to the game that involve elevating your priveleges within the web app, eventually leading to remote code execution (RCE). WARNING: Due to the nature of this application, it is not intended to be used as a functional website. To avoid damaging files or putting your PC at risk, the server should be run on a virtual machine that is not exposed to the internet.

LEVEL 1

The first challenge requires poking and proding to find what you're looking for... You must locate a resource uniformly, you will know when you find it and if you can't find the resource you can always go bust. The answer is obvious.

Access Control is an important part of web app security, although it isn't the most common type of vulnerability, it is considered Top 10 web app vulnerabilities in 2017 by OWASP. Don't overthink it, if you're a beginner you might want to try guessing.
